# Webapp — Amazon Review Analyzer

**Project 3: NLP Business Case** | Group 4 | IronHack AI Engineering Bootcamp
**Team:** Krzysztof Giwojno & Felipe Doria

Static single-page web app for browsing NLP project results.

## Structure

```
webapp/
  index.html              # Single-page app (HTML + CSS + JS, no framework)
  generate_webapp.py      # Script to generate data/ from project outputs
  deploy.sh               # Upload script for OVH hosting
  data/                   # Generated JSON files (not in git, generated by script)
    stats.json            # Dashboard overview numbers
    clusters.json         # Cluster summaries + metadata
    products.json         # Product data + AI summaries
    reviews_sample.json   # Sample reviews with model predictions
```

## Setup

### 1. Generate data files

Run from the project root (where your CSV and JSON files are):

```bash
python webapp/generate_webapp.py
```

This reads:
- `data_with_clusters.csv`
- `data_with_predictions_v2.csv`
- `summaries_api.json`
- `product_clusters.csv`

And produces `webapp/data/*.json`

### 2. Test locally

Open `webapp/index.html` in a browser, or serve with Python:

```bash
cd webapp
python -m http.server 8000
# Open http://localhost:8000
```

Note: You must use a local server (not file://) because the app loads JSON via fetch().

### 3. Deploy to OVH

Edit `deploy.sh` with your OVH credentials, then:

```bash
chmod +x deploy.sh
./deploy.sh
```

Or manually upload via FTP:
- Upload `index.html` to your web root
- Upload the `data/` folder alongside it

## Features

- **Dashboard** — Overview stats, sentiment distribution, rating charts, reviews by category
- **Categories** — Browse 6 product clusters with AI-generated recommendation articles
- **Products** — Filter and browse all 65 products, click for AI summaries
- **Reviews** — Explore sample reviews with filters (category, sentiment, rating, search)
- **About** — Model details, project info

## No backend required

Everything is pre-computed and served as static JSON. The app works on any hosting that serves HTML files — no Python, no database, no API keys needed at runtime.

## Tech

- Pure HTML/CSS/JS, no framework dependencies
- Google Fonts (DM Sans + DM Serif Display)
- Dark theme, responsive layout
- ~15KB HTML + generated JSON data files
